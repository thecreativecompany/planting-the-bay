# Planting the Bay — Vercel Static Site

This package keeps the original/default Webflow design and updates the website content/details to match the Planting the Bay brief.

## Vercel settings

- Framework Preset: Other
- Install Command: npm install
- Build Command: npm run vercel-build
- Output Directory: dist

## Notes

- Design/classes/assets from the original Webflow export are preserved.
- New pages reuse the default project-detail page layout from the template.
- Replace the placeholder visuals in `/images` with final Bay Area photography, video stills, logo/wordmark, and the 3D Bay Area flyover later.
- Forms and giving buttons are placeholder/static and need connection to the final form/giving provider.
- The design brief is stored in `_project_docs/`.
